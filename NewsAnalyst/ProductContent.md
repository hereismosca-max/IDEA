# Product Content

## 产品是什么 What is this product?

**NewsAnalyst** 是一个面向经济金融领域的新闻聚合与分析平台。

它从多个权威英文财经媒体（Reuters、CNBC、AP News、Yahoo Finance、MarketWatch）自动抓取新闻，经过筛选过滤后，以整洁的卡片形式呈现给用户。平台的核心理念是：**去除噱头，回归事实**——拒绝标题党，只呈现客观、有价值的经济金融资讯。

未来版本将接入 AI，对新闻进行自动摘要、分类打标签、重要性评分，帮助用户在信息爆炸的环境中快速获取真正有价值的内容。

---

## 为什么做这个产品 Why build this?

当前的财经新闻生态存在几个明显的问题：

1. **标题党泛滥**：大量媒体为了流量故意夸大、煽情，导致读者难以做出理性判断。
2. **信息过载**：每天产生的财经新闻数量庞大，人工筛选耗时耗力。
3. **缺乏客观摘要**：大多数资讯平台只是转载原文，缺少简洁中立的内容提炼。
4. **个人学习需求**：创始人希望通过系统性地观察市场动态，提升对经济金融的理解，辅助基金和股票类投资决策。

除了产品本身的实用价值，本项目同时也是一个完整的工程实践项目，目标是积累从 0 到 1 构建全栈产品的项目经验。

## 目标用户 Target Users

**第一阶段（当前）**：创始人本人
- 具备一定经济金融基础知识
- 希望通过学习和观察市场进行基金、股票类投资
- 需要高质量、客观中立的财经资讯，而非情绪化内容

**未来阶段（扩展）**：
- 有一定金融知识储备、希望自主做投资决策的个人投资者
- 不满足于主流媒体标题党内容、追求信息质量的读者
- 关注宏观经济、市场动态的学习者和从业者

## 产品最终形态 Final Product Vision

### 核心功能
- **新闻聚合**：自动抓取多个权威财经媒体，每 6 小时更新一次
- **智能筛选**：AI 过滤无关内容，只保留经济金融相关新闻
- **客观摘要**：AI 生成不带立场的一句话/一段话客观摘要
- **自动分类**：AI 自动打标签（Markets、Economy、Policy、Stocks 等）
- **重要性评分**：AI 对每条新闻评定价值权重，重要新闻优先展示
- **双语支持**：英文板块（美国市场）+ 中文板块（中国市场），语言可切换

### 界面形态
- 顶栏：网站名称 + 导航 + 语言切换 + 用户账户
- 横向菜单栏：按分类浏览（All / Markets / Economy / Policy / Stocks / Commodities / Crypto）
- 新闻卡片流：标题、来源、时间、客观摘要、分类标签、跳转原文

### 技术特征
- 全栈 Web 应用，公开部署，有独立 URL
- 用户系统：注册、登录、收藏新闻
- 模块化设计，持续迭代，长期维护
